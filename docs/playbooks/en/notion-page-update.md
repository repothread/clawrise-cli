# Update a Notion Page

Operations:

- `notion.page.update`
- `notion.page.markdown.update`
- `notion.page.get`
- `notion.page.markdown.get`

## Read the page first

```bash
clawrise notion.page.get --json '{"page_id":"page_demo"}'
clawrise notion.page.markdown.get --json '{"page_id":"page_demo"}'
```

## Update title or properties

```bash
clawrise notion.page.update --dry-run --json '{
  "page_id":"page_demo",
  "title":"Updated project notes"
}'
```

## Replace markdown content

```bash
clawrise notion.page.markdown.update --dry-run --json '{
  "page_id":"page_demo",
  "type":"replace_content",
  "replace_content":{
    "new_str":"# Updated title\n\nGenerated by Clawrise."
  }
}'
```

# Update a Feishu Document

Operations:

- `feishu.docs.document.edit`
- `feishu.docs.document.get`
- `feishu.docs.document.get_raw_content`

Use this playbook when you want to replace a document body or append new blocks.

## Read the current content first

```bash
clawrise feishu.docs.document.get_raw_content --json '{"document_id":"doxcnDemo"}'
```

## Replace the whole document

```bash
clawrise feishu.docs.document.edit --dry-run --json '{
  "document_id":"doxcnDemo",
  "mode":"replace_all",
  "blocks":[
    {"type":"heading_1","text":"Weekly notes"},
    {"type":"paragraph","text":"Generated by Clawrise."}
  ]
}'
```

Remove `--dry-run` for the real write.

## Append one paragraph

```bash
clawrise feishu.docs.document.edit --json '{
  "document_id":"doxcnDemo",
  "mode":"append",
  "blocks":[
    {"type":"paragraph","text":"One more paragraph."}
  ]
}'
```
